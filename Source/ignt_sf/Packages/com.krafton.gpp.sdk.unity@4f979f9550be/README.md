# Krafton GPP Unity SDK

## Getting Started

### Preparation

1. Using the Unity Package Manager, "+" ->  "Add Package from Git URL" -> Paste in https://github.krafton.com/gpp/gpp-unity-sdk.git?path=/com.krafton.gpp.sdk.unity" -> "Add"
2. Use the "GPP/Edit Settings" button to open the GPP SDK Configuration tool. Set all relevant fields. When you press save, this will generate a file in your Resources folder.
